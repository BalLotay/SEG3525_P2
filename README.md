# Horizon Studio

Horizon Studio is a multi-page photography studio website built for SEG3525 (uOttawa).
It includes services, photographers, location/contact info, and a 3-step booking flow.

## Technologies

- HTML5
- CSS3
- Bootstrap 5
- Vite

## Installation

### Prerequisites

- Node.js 18+
- npm

### Run locally

```bash
npm install
npm run dev
```

### Production build

```bash
npm run build
npm run preview
```
